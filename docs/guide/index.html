# Introducción

Este material guiará a la realización de documentación técnica usando vuepress (opens new window) el cual tiene su propio HTML (opens new window) estático renderizado previamente, lo que brinda un excelente rendimiento de carga y es compatible con SEO. Sin embargo, una vez que se carga la página, Vue (opens new window) se hace cargo del contenido estático y lo convierte en una aplicación de página única SPA (opens new window) completa.

Con lo cual podemos redactar todo tiempo de documentación basada en una estructura en la web inclusive se nos facilitara pasar ya contenidos en otros documentos que puedan ser visualizados como un contenido web, favoreciendo a una interfaz más amigable.

Recomendación

Esta guía se realizó para poder acompañar y ayudar hacer documentación con esta herramienta de todos modos se recomienda ver la documentación de la página oficial de Vuepress (opens new window).
